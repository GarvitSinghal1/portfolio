# Garvit Singhal - Portfolio Website

A futuristic, single-page portfolio website designed to showcase projects, research, and photography. Built with a focus on immersive user experience, featuring glassmorphism design, particle animations, and 3D interactions.

## üöÄ Features

*   **Single-File Architecture**: The entire website (HTML, CSS, JS) is contained within a single `index.html` file for simplicity and ease of deployment.
*   **Immersive Hero Section**: Interactive particle background and typing text animation.
*   **Glassmorphism Design**: Modern, translucent UI elements with blur effects.
*   **Smooth Animations**: Scroll reveal animations and 3D tilt effects on project cards.
*   **Photography Gallery**: A dedicated section for photography with a custom lightbox viewer.
*   **Responsive**: Fully responsive design that adapts to mobile, tablet, and desktop screens.
